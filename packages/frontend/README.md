# Onder Frontend

## Build Setup

``` bash
# install dependencies
yarn

# build for production with minification
yarn build

# build single system for production
## hardware
yarn build-hardware
## conusmer
yarn build-consumer
## operator
yarn build-operator

# serve with hot reload at localhost:8080
## hardware
yarn dev-hardware
## conusmer
yarn dev-consumer
## operator
yarn dev-operator

# build for production and view the bundle analyzer report
yarn <build-config> --report
```
